    TinyNOS — Спецификация функций¶

    Сводная матрица возможностей для RFI-раунда и сравнения с эталонными NOS. Полный продуктовый обзор — в Datasheet; как настраивать — в Руководстве по настройке.

    Легенда статусов

    Метка Значение
    GA Generally Available — валидировано end-to-end на программной / lab-фабрике
    Preview Реализовано и пригодно к использованию; валидация на оборудовании ещё ожидается
    Planned Заявленная цель roadmap v1.1

    Дисциплина честности: ни одна строка не претендует на ASIC-offload на реальном кремнии или GA на production-оборудовании — эти пункты явно помечены и в работе для v1.1.

    Layer 2 — коммутация¶

    Функция Что даёт Статус
    VLAN 802.1Q Trunk / access / native VLAN; диапазон 1–4094 GA
    Q-in-Q (802.1ad) Стэкированная VLAN-инкапсуляция для provider bridging Planned
    Spanning Tree (STP/RSTP/MSTP) 802.1D/w/s; multi-region MSTP, быстрый edge-переход, защита BPDU Preview
    Агрегация каналов (LACP) 802.3ad/802.1AX; fast/slow rate; хеш-политики L2/L3/L3+4 GA
    MLAG (multi-chassis LAG) Двойно-активная избыточность top-of-rack; защита от split-brain GA
    LLDP / LLDP-MED 802.1AB; обнаружение соседей; voice-VLAN/network-policy GA
    Storm control Per-interface лимит broadcast / multicast / unknown-unicast GA
    Зеркалирование портов (SPAN) Локальный port mirror (зависит от модулей ядра) Preview
    Jumbo-фреймы MTU до 9216 на интерфейсах, мостах и SVI GA
    Статические MAC (FDB) Постоянные FDB-записи на VLAN; статические VXLAN-туннели GA
    MAC mobility EVPN Type-2 с sequence-number; детект дублей хоста GA

    Layer 3 — IP-маршрутизация¶

    Функция Что даёт Статус
    Статическая маршрутизация (v4/v6) Per-VRF; blackhole; distance/metric; рекурсивный next-hop; ECMP/UCMP-веса GA
    BGP (RFC 4271) iBGP/eBGP; route-reflection; конфедерации; 4-байтный ASN GA
    BGP communities Standard / extended / large (RFC 1997 / 4360 / 8092) GA
    BGP unnumbered IPv6 link-local peering с обменом IPv4 NLRI — zero-touch leaf-spine GA
    BGP graceful restart RFC 4724; NSF-aware сохранение маршрутов GA
    BGP add-path RFC 7911; multi-path-анонс на префикс GA
    OSPFv2 RFC 2328; areas, NSSA, virtual links, MD5/SHA, интеграция с BFD GA
    OSPFv3 RFC 5340; IPv6 + IPv4 address-family Preview
    IS-IS RFC 1142/5301; L1/L2; multi-topology IPv6; BFD; TI-LFA fast-reroute GA
    BFD RFC 5880; single/multi-hop; субсекундное обнаружение отказов GA
    ECMP / UCMP Per-VRF равно- и неравностоимостные пути; resilient hashing GA
    Policy-Based Routing Маршрутизация по политике (match/action, per-VRF lookup) Preview
    Route-maps Match (prefix-list/community/AS-path); set (community/local-pref/metric) GA
    Multicast (PIM-SM/SSM) Anycast-RP; MSDP peer-mesh; IGMP/MLD snooping Preview

    Сетевая виртуализация и оверлей¶

    Функция Что даёт Статус
    VRF Жёсткая изоляция тенантов; per-VRF маршрутизация GA
    Route-leaking между VRF Контролируемый импорт/экспорт через route-target GA
    VXLAN (RFC 7348) L2-оверлей; UDP-4789 по умолчанию; настраиваемый порт GA
    EVPN Type-2 RFC 7432; анонс MAC/IP через BGP GA
    EVPN Type-3 Inclusive Multicast для BUM (ingress / egress / PIM-SM) GA
    EVPN Type-5 RFC 9136; анонс IP-префиксов; symmetric IRB GA
    EVPN-MH (all-active) RFC 7432; Ethernet Segment; выбор designated-forwarder GA
    Anycast gateway Общий шлюз на каждом leaf; symmetric IRB GA
    Symmetric / asymmetric IRB Маршрутизация интегрированно с мостом, оба режима GA
    Single-VXLAN-Device (SVD) Один vxlan0 на все VNI; требует kernel ≥ 5.18 GA
    Inter-AS EVPN RFC 9135; сохранение originating VTEP через eBGP GA
    SR-MPLS Segment routing по MPLS-стеку меток Preview
    SRv6 (uSID, TI-LFA) RFC 8754/8986; IPv6-нативный segment routing с fast-reroute GA
    SRv6 L3VPN per-VRF End.DT4/DT6 service-SID из BGP Prefix-SID GA

    Высокая доступность¶

    Функция Что даёт Статус
    Транзакционный commit 6 стадий: проверка → превью → применение → самопроверка GA
    Атомарный rollback Любой сбой откатывает изменение начисто, без частичного состояния GA
    Commit-confirmed Авто-откат по таймеру, если оператор не подтвердил GA
    Checkpoints и revisions Именованные снапшоты и полная история с атрибуцией оператора GA
    BGP Graceful Restart RFC 4724/8538; непрерывность при перезапуске control plane GA
    Non-Stop Forwarding Зависит от аппаратного offload Preview
    MLAG peer-link FDB sync Синхронизация таблиц между парой; полная пересинхронизация при rejoin GA
    EVPN-MH split-horizon All-active multihoming без петель; выбор DF GA
    Anycast VTEP Loopback, общий на MLAG-пару; сходимость ECMP underlay GA
    ISSU-оркестратор In-service upgrade с health-check gating Planned

    Референсные NOS

    Сопоставимый объём и поверхность функций: Cisco Nexus NX-OS · Arista EOS · NVIDIA Cumulus Linux · IP Infusion OcNOS · SONiC. TinyNOS целится в функциональную parity на production-критичной поверхности control- и dataplane, оставаясь open source под Apache-2.0 без vendor lock-in.

    Все названия продуктов и бренды — собственность соответствующих владельцев; упоминание не подразумевает одобрения.
